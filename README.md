# Multi-Phase Analytics Dashboard

A Dash-based analytics dashboard covering stock market analysis, ML models, and NLP sentiment analysis.

## Files Required
- `app.py` — Main dashboard application
- `df.csv` — Primary dataset
- `web_scrape.csv` — Web scraped news data
- `x_train2_5.csv`, `y_train2_5.csv` — Training data
- `x_test2_5.csv`, `y_test2_5.csv` — Test data

## Local Run
```bash
pip install -r requirements.txt
python app.py
# Open http://localhost:8050
```
